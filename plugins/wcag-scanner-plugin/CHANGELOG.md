# Changelog

All notable changes to the WCAG Scanner Plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-17

### Added
- Initial release of WCAG Scanner Plugin
- `/scan-wcag` command for explicit WCAG 2.2 compliance scanning
- Accessibility Reviewer agent for comprehensive accessibility audits
- WCAG Scan skill for autonomous accessibility issue detection
- PostToolUse hook for automatic accessibility checks after file writes
- Support for HTML, CSS, JavaScript/TypeScript, JSX, TSX, Vue, Svelte, and Astro files
- WCAG 2.2 Level A, AA, and AAA compliance checks
- Intelligent issue categorization (Critical, Serious, Moderate, Minor)
- Multiple output formats (text, JSON, HTML)
- Watch mode for continuous scanning during development
- Framework-specific analysis for React, Vue, Angular, and Svelte
- Color contrast ratio verification
- Semantic HTML evaluation
- ARIA attribute validation
- Form accessibility checking
- Keyboard navigation analysis
- Screen reader compatibility guidance
- Actionable remediation guidance with code examples
- Comprehensive documentation and examples
