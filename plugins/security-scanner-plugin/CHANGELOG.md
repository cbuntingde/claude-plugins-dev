# Changelog

All notable changes to the Security Scanner Plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-17

### Added
- Initial release of Security Scanner Plugin
- OWASP Top 10 vulnerability detection
- Three slash commands: `/security-scan`, `/vulnerability-check`, `/dependency-audit`
- Three security agents: Vulnerability Detector, Security Auditor, Code Reviewer
- Three autonomous skills: Security Analyzer, Dependency Scanner, Config Validator
- Automated security hooks for real-time vulnerability detection
- Support for multiple programming languages (JS/TS, Python, Java, Go, PHP, Ruby, C#)
- Support for multiple package managers (npm, pip, bundler, composer, maven, go, cargo)
- Configuration validation for security misconfigurations
- Hardcoded secrets detection
- Auto-fix suggestions for vulnerabilities
- JSON and text output formats
- CI/CD integration support
- Custom security policies configuration
- Comprehensive documentation
