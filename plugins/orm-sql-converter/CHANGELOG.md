# Changelog

All notable changes to the ORM-SQL Converter plugin will be documented in this file.

## [1.0.0] - 2025-01-17

### Added
- Initial release of ORM-SQL Converter plugin
- Support for converting ORM queries to raw SQL
- Support for converting raw SQL to ORM queries
- Query explanation and analysis feature
- ORM-SQL Specialist agent
- ORM Converter skill
- Support for 10+ ORM frameworks across 6+ programming languages
- Support for 6+ database systems
- N+1 query detection
- Performance optimization recommendations
- Window function support
- Common Table Expression (CTE) support
- Index recommendations
- Comprehensive examples and documentation
