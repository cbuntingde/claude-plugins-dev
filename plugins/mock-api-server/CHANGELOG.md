# Changelog

All notable changes to the Mock API Server plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-17

### Added
- Initial release of Mock API Server plugin
- `/mock-api` slash command for server management
- Mock API Designer agent for complex scenarios
- mock-api skill for automatic invocation
- MCP server integration for advanced control
- JSON-based configuration system
- Support for dynamic path parameters
- Template variable support (params, query, body)
- Configurable response delays
- Multiple simultaneous servers
- Request logging
- CORS support
- Example configurations:
  - Todo API
  - User management API
  - Error scenarios API
- Comprehensive documentation and README

### Features
- Quick server startup in seconds
- Flexible endpoint configuration
- Realistic response generation
- Error simulation with custom status codes
- OpenAPI/Swagger import capability (via agent)
- Status monitoring for all running servers
- RESTful API design support
- HTTP method support (GET, POST, PUT, DELETE, etc.)
