# Changelog

All notable changes to the Performance Profiler plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-17

### Added
- Initial release of Performance Profiler plugin
- `/profile` command for code execution profiling
- `/benchmark` command for comparative performance testing
- `/optimize` command for automatic optimizations
- `/analyze-memory` command for memory leak detection
- `/compare-perf` command for performance comparison
- Performance Analyst agent for deep analysis
- Performance Analyzer skill for automatic detection
- Post-write hooks for performance analysis
- Multi-language support (JavaScript/TypeScript, Python, Go, Java, Rust)
- Tool installation script for profiling dependencies
- Baseline generation script
- Comprehensive documentation

### Features
- Hot spot identification
- Call graph analysis
- Memory leak detection
- Database query optimization (N+1 detection)
- Caching strategy suggestions
- Async/await optimization
- Algorithm complexity analysis
- CI/CD integration support
- Performance regression detection
- Trend analysis and visualization
