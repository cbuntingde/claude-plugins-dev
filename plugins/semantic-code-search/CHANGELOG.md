# Changelog

All notable changes to the Semantic Code Search plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-17

### Added
- Initial release of Semantic Code Search plugin
- `/semantic-search` command for natural language code queries
- `/semantic-index` command for building vector embeddings index
- Semantic Code Explorer agent for autonomous code discovery
- MCP server with semantic search capabilities
- Vector-based search using cosine similarity
- Context-rich results with relevance scoring
- Support for multiple programming languages
- Incremental and full index rebuilding
- Skill for automatic semantic code discovery

### Features
- Natural language query understanding
- Semantic similarity matching
- Pattern recognition across codebases
- Code relationship mapping
- Implementation finding without exact keywords
- Context-aware search results
- File preview and snippet extraction

- Comprehensive README with usage examples
