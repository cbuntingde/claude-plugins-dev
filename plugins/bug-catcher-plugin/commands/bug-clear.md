# Bug Clear

Clears all captured tool errors from the bug log.

## Usage

```
/bug-clear
```

## Description

This command removes all captured bugs from the log file. Use this after reviewing and fixing the captured errors to start fresh.

## Examples

```
/bug-clear
```

Output:
```
Bug log cleared.
```

## Related Commands

- `/bug-list` - Display captured bugs
