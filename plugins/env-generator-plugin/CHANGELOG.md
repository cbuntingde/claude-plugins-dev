# Changelog

All notable changes to the "env-generator" plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-17

### Added
- Initial release of env-generator plugin
- `/generate-env` command for generating .env files from code analysis
- `/validate-env` command for validating .env files
- Environment Variable Analyzer agent for deep code analysis
- env-detect skill for automatic environment variable detection
- Support for multiple programming languages (JavaScript, Python, Go, Ruby, PHP, Java, C#, Shell)
- Smart categorization of environment variables
- Type detection and validation
- Security awareness for sensitive variables
- Comprehensive documentation and examples
